## Descrição

Aplicação web desenvolvida em Angular para gerenciamento de oficinas mecânicas, permitindo o controle de clientes, veículos, ordens de serviço e usuários.

---

## Tecnologias Utilizadas

- Angular 21
- TypeScript
- HTML5
- CSS3
- Firebase Hosting

---

## Funcionalidades

- Login com controle de acesso
- Dashboard com estatísticas
- Cadastro de clientes
- Cadastro de veículos
- Gerenciamento de ordens de serviço
- Relatórios gerenciais
- Integração com API Spring Boot

---

## Como Executar

### Pré-requisitos

- Node.js 20 ou superior
- Angular CLI

### Instalação

```bash
npm install
```

### Execução

```bash
ng serve --port 4200
```

Aplicação disponível em:

```text
http://localhost:4200
```
